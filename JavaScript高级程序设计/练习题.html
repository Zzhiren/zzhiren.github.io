# 练习题

# 下面函数的最终打印顺序

function test() {
  setTimeout(function(){
      console.log(1)
  }, 0);
  new Promise(function(a, b){
      setTimeout(function(){
          for(var i=0; i< 10 ; i++){
            i == 9 && a()
          }
      }, 2000);
      console.log(2)
  }).then(function(){
    console.log(3)
  })catch(function(){
  });
  console.log(4)
}


function test2(){
  setTimeout(function(){
      console.log(1)
  }, 0);
  console.log(2)
}

解答

setTimeout 的时间参数最低为4ms，浏览器会将0改为4ms；js为单线程执行，当第一个setTimeout为延时执行后，js会立即执行后面的代码，此时执行到new Promise，其中的a和b方法就是promise的两个回调resolve（成功后执行的方法，then）和reject（错误后执行的方法catch）// 2,4,1,3
